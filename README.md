# Castro Solutions Homebrew Tap

Homebrew casks for [Castro Solutions](https://github.com/castrosolutions) apps.

## Install

```sh
brew install --cask castrosolutions/tap/chacharapp
```

## Available casks

| Cask | App | Description |
| --- | --- | --- |
| `chacharapp` | [ChacharApp](https://github.com/castrosolutions/ChacharApp) | Local, free, private voice dictation for macOS (Apple Silicon) |

## Uninstall

```sh
brew uninstall --cask chacharapp        # removes the app
brew uninstall --zap --cask chacharapp  # also removes settings, vocabulary, history and downloaded models
```
